# Day 01 - Interview Questions

## Basic Questions

### 1. What is JavaScript?

JavaScript is a high-level, interpreted programming language used to create interactive web applications.

---

### 2. Where can JavaScript run?

- Web Browsers
- Node.js

---

### 3. What is console.log()?

It is used to print output to the console.

---

### 4. What are variables?

Variables are named containers used to store data.

Example

```javascript
let name = "Robin";
```

---

### 5. Why do we use variables?

Variables help us store, update, and reuse data in a program.

---

### 6. What are the rules for naming variables?

- Start with a letter, `_`, or `$`
- Cannot start with a number
- Cannot use reserved keywords
- Case-sensitive

---

### 7. What are Primitive Data Types?

- String
- Number
- Boolean
- Undefined
- Null
- BigInt
- Symbol

---

### 8. What is the typeof operator?

It is used to check the data type of a value.

Example

```javascript
typeof "Robin"; // string
```

---

### 9. What is the difference between null and undefined?

**undefined**
- Variable is declared but not assigned a value.

**null**
- Represents an intentional empty value.

---

### 10. What are comments in JavaScript?

Comments are ignored by the JavaScript engine and are used to explain code.

Single-line:

```javascript
// Comment
```

Multi-line:

```javascript
/*
Comment
*/
```

---

### 11. Name some console methods.

- console.log()
- console.info()
- console.warn()
- console.error()
- console.table()

---

### 12. What is camelCase?

A naming convention where the first word starts with a lowercase letter and every next word starts with an uppercase letter.

Example

```javascript
let firstName;
let employeeSalary;
```

---

# Interview Tip

Most freshers and early-experience interviews begin with these questions:

- What is JavaScript?
- Difference between Java and JavaScript?
- What are variables?
- What are data types?
- Difference between null and undefined?
- What is typeof?
- What is console.log()?
